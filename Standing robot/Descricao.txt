Este projeto consiste em fazer um robot de duas rodas que se equilibre sozinho. Para tal usei o sensor de 6 eixos (giroscópio e acelerômetro) e usei o filtro de kalman para fazer o datafusion. O desenho 3D foi feito em Blender e algumas partes são assets que se podem encontrar na internet de graça. O site que usei para estes foi o Thingsverse. O microcontrolador é o Arduino uno porque, apesar de não ter muita capacidade computacional e ser algo robusto, era o único disponível e acreditava que funcionaria apesar destes problemas. A calibração foi feita de forma bastante manual com um código extremamente simples dado que não planeava mudar o sensor e apenas uma calibração é necessária. Ainda assim o código encontra-se disponível no repositório. Para controlar os motores foi usada uma H-bridge e para aumentar a corrente usou-se uma pilha de 9V. A pilha também serviu para alimentar o Arduino. Para facilitar o manuseamento do robot colocou-se um switch que cortava a corrente da pilha. Para controlar os motores usou-se um PID com uma constante derivativa grande para obter estabilidade inicial sem encoders.
